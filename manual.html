S.E.T. Device (Firmware v63.0)

1. Device Overview

The S.E.T. is a dedicated ultrasonic anomaly detector designed for paranormal investigation. It utilizes active echolocation (sonar) to detect physical mass in real-time. Firmware v63.0 introduces a Dual-Mode Operation system, allowing for both stationary trap deployment and handheld sweeping.

Key Features:

  • Active Echolocation: Continuous ultrasonic pinging to detect physical anomalies.
  • Dual-Mode Boot: Switch between Stationary Trap and Handheld Sweep on startup.
  • Dynamic Scaling: Visual tracking bars adapt automatically to the environment.
  • Odometer: Tracks total device runtime (Lifetime Hours) in internal memory.

2. Smart Boot System (Mode Selection)

The device uses a single-button "Elevator" system to select your operating mode during power-on.

A. Stationary Mode (Default)

  • Action: Flip the power switch ON. Do not touch any buttons.
  • Function: The device performs a 3-stage wall calibration and enters "Trap Mode." Ideally used for monitoring doorways or hallways.

B. Handheld Mode (Sweep)

  • Action: Hold the RCL Button while powering on.
  • Release: Let go when the screen says RELEASE FOR HANDHELD (approx. 1 second).
  • Function: Skips calibration and enters "Active Sweep." The sensor becomes highly sensitive to any object within 6 feet.

C. Diagnostics Mode

  • Action: Keep holding the RCL Button past the handheld prompt.
  • Release: Let go when the screen says HOLD FOR DIAGNOSTICS (approx. 4 seconds).
  • Function: Enters the system maintenance menu.

3. Scanning Operations

Mode A: Stationary Trap

CRITICAL: In Stationary Mode, the device creates a "Safety Buffer" of 6 inches from the calibrated wall. Any object entering the space between the device and the buffer will trigger the alarm.
  • Visuals: Displays a Dashboard with Battery, Session Hits, and Lifetime Hits.
  • Alert: When an anomaly is detected, the screen flashes ANOMALY DETECTED and freezes for 2 seconds for evidence review.
  • Tracking: After the freeze, it enters "Tracking Mode" with a dynamic bar graph.

Mode B: Handheld Sweep

  • Visuals: Displays a large "Live Gauge" and distance readout. There is no dashboard or waiting period.
  • Audio (Geiger Style): The device emits clicking sounds that increase in speed as you get closer to an object or anomaly.
  • Range: Fixed to 72 inches (6 feet). The bar graph fills up as you approach a target.

4. Data Logging & Recall

The device stores distance and timestamp data for the last 20 anomalies detected in the current session.

  • Enter Logs: Press the RCL Button once during scanning (Stationary Mode only).
  • Navigation: Press RCL again to scroll to the next log entry.
  • Exit to Scanning: HOLD the RCL Button for 1.5 seconds until the screen returns to scanning mode.

5. Diagnostics & Maintenance

Access this menu via the "Smart Boot" system (Hold button for 4s on startup).

  • Page 1 (Reset): Hold the button to wipe "Lifetime Hits" and "Runtime" from memory.
  • Page 2 (LED Test): Flashes the front notification LED.
  • Page 3 (Battery): Displays raw voltage input.
  • Page 4 (Odometer): Displays total device runtime in Hours/Minutes. Note: Runtime saves to memory every 10 minutes to preserve chip life.
