S.E.T. Device (Model v1.0)

1. Device Overview

The S.E.T. is a dedicated ultrasonic anomaly detector designed for paranormal investigation. Unlike standard EMF meters, this device uses active echolocation (sonar) to map the physical distance to a target wall or object. It monitors this baseline for deviations, alerting the operator only when a physical mass manifests within the calibrated zone.

Key Features:

  • Active Echolocation: Continuous ultrasonic pinging to detect physical anomalies.
  • Smart Calibration (v32): 3-Stage stability audit with Infinity Rejection to prevent false positives.
  • Anomaly Freeze: Instant "Snapshot" of distance with a 2-second visual hold for evidence review.
  • Session Logging: Internal memory stores the last 20 events of the current session.

2. Startup & Calibration

CRITICAL: The device must be calibrated against a solid, stationary surface (wall, door, etc.) OR pointed at open space before scanning can begin.
  1. Power On: Flip the main power switch.
  2. The "Smart Calibration" Sequence: The device will begin active scanning immediately. It performs a 3-Strike Stability Check. It must receive consistent readings for 3 consecutive cycles to lock.

Status Indicators: CALIBRATING... → VERIFYING... → CONFIRMING...

Calibration Notes:

  • Open Space Mode: If no wall is detected during startup, the device defaults to a maximum range of 120 Inches.
  • ERROR: UNSTABLE AREA / RE-AIM DEVICE
    Cause: The sensor is detecting "jumping" numbers (e.g., the beam is hitting the edge of a chair or railing).
    Solution: Move the device 1â€“2 inches left or right to find a solid surface.

3. Scanning Operation

Once RANGE CALIBRATED appears, the device enters Active Scan Mode.

The Scanning Screen Layout:

  • Top Row: SCANNING... (Active Status).
  • Middle Left: TOTAL: X (Total Lifetime Hits).
  • Middle Right: SESS: X (Current Session Hits).
  • Bottom Center: LAST: X (Distance of the last recorded anomaly).

Detection Trigger:

If an object (or anomaly) is detected closer than the calibrated wall distance (minus 5 inches), the alarm triggers.

Phase 1: The Alert (0 to 2 Seconds)
Audio: Sharp Double-Beep.
Visual: Screen displays ANOMALY DETECTED and distance.
Behavior: The screen FREEZES for 2 seconds for evidence review.

Phase 2: Live Tracking (> 2 Seconds)
If the anomaly remains, the screen switches to Tracking Mode.
Visual: Spectral avatar (Ghost) appears with live distance readout.

4. Data Logging & Recall

The device stores distance and timestamp data for the last 20 anomalies detected in the current session.

  • Enter Logs: Press the RCL (Recall) button once during scanning.
  • Navigation: Press RCL again to scroll to the next log entry.
  • Exit / Reset: Press the RST (Reset) button to immediately return to Active Scanning.

5. Diagnostics & Reset

To access the hidden diagnostics menu:

  1. Hold the RCL Button WHILE powering on the device.
  2. Release when DIAGNOSTICS appears.
  • Wipe Data: Hold the RCL Button when prompted to reset "Lifetime Hits".
  • Hardware Test: Cycles LED, Buzzer, and Sensor.
  • Battery Check: Displays raw voltage.
